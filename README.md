# comment-service
评论服务
### 编写缘由：建立自己的墙内插件
网站是jekyll平台搭建的，纯静态网站，就需要第三方插件，通过js来增加评论功能。
国内评论系统大多数停止服务了，有少数就比较麻烦需要备案，认证等一些，就没有去找那些可用好用的服务了；国外的因为**互联网国情**就不采用了。

于是就有了自己来搭建一套评论服务也正好来练练手，也为了熟悉之前未接触过的一些业务等。

最开始规划当前从最简单开始，版本是一代一代迭代升级的。

---
### 最基本需求功能
> * 帖子发表评论
> * 帖子评论展示
> * 回复评论
